Analysis Template Builder

This template is for any task where a source text is provided, and there are instructions and a format provided to perform that task.

The form fields below allow you to select from one of the pre-made templates, make any custom alterations and copy the completed prompt.


Description of Fields

Role: Uses 'role-casting', specifying a human role or even naming a specific person.
Task: A sentence fragment briefly describing the overall goal, used at top and before instructions.
Instructions: More detailed instructions for how task should be completed. Aim for positive rather than negative instructions. Point form or numbered instructions can be helpful.
Format: Demonstrates heading levels, table formatting, or any other specifications, usually using Markdown. May also include examples or further instructions.
Description of source: Sentence fragment providing context for what the source text is.
Source text: Source text or excerpts from multiple texts. If multiple texts, clearly delineate each and provide a citation. Due to context window, should be limited to about 1500-2000 words at maximum, but shorter is more effective.
